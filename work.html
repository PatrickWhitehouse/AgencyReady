My Work

Here are some examples of projects I've been involved in. They are either university related or personal.

DPM Project

The task was to create a practical website for students which provided useful information. This included: places to shop, places to eat and places to visit within Manchester. My role within this project was lead web developer. I had the fortunate task of creating the website based on research from other group members.

View DPM Project

Maid About Town

This was a final year project. The website was created for a commercial cleaners based in Cheshire.
I had to carry out my own research in order to develop the website, such as: a competitor analysis, card sort and a selection of user persona mockups.

View Client Project

Web Programming Restaurant Menu

The task was to create a menu for a restaurant. The menu included a search function which retrieved data from JSON files. I also implemented other features such as using the date function to automatically load the correct menu for that day.
This was my first experience using AJAX requests in order to get information from JSON files.

View Menu Project

JavaScript Order Form

This project was to use the basics of JavaScript in order to create a fully functioning ordering form for a clothing company. The project consisted of using variables and basic math functions to calculate order prices. I also had to use some advanced math methods in order to calculate discounts based on whether the customer was registered or not.
